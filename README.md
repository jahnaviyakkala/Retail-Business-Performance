# Retail Business Performance & Profitability Analysis

## Objective
Analyze retail transactional data to identify profit-draining categories,
optimize inventory levels, and understand seasonal sales behavior.

## Tools Used
- SQL (SQLite) – data cleaning & aggregation
- Python (Pandas, Matplotlib) – correlation & trend analysis
- Power BI – interactive dashboard

## Key Insights
- High sales do not always translate to high profit (Furniture category)
- Office Supplies shows highest profit efficiency
- Inventory quantity shows weak correlation with profit
- Clear seasonal profit trends observed

## Files
- queries.sql – SQL queries used for analysis
- retail_analysis.ipynb – Python analysis
- retail_dashboard.pbix – Power BI dashboard
- Retail_Business_Performance_Report.pdf – Final report
